## Quick Start

### Prerequisites
- Python 3.11+
- pip

### Installation

```bash
# Install dependencies
pip install -r requirements.txt

# Copy environment file
cp src/.env.example src/.env

# Run the server
uvicorn src.main:app --reload --port {{PORT}}
```

The server will start at `http://localhost:{{PORT}}`

### API Documentation
- **Swagger UI**: http://localhost:{{PORT}}/docs
- **ReDoc**: http://localhost:{{PORT}}/redoc

## Endpoints

- `GET /healthz` - Health check
- `GET /api/users/{user_id}` - Fetch single user
- `GET /api/users/{user_id}/posts` - Fetch user's posts
- `GET /api/users/{user_id}/aggregate` - Aggregate user + posts (parallel)

## Docker

```bash
# Build image
docker build -t {{PROJECT_NAME}}-bff .

# Run container
docker run -p {{PORT}}:{{PORT}} --env-file src/.env {{PROJECT_NAME}}-bff
```

## Configuration

- `BACKEND_API_URL` - Backend service URL
